# 🎵 Music Player Web Application

## 📌 Project Overview
This project is a simple and interactive **Music Player Web Application** developed using HTML, CSS, and TypeScript (compiled into JavaScript).  

It is designed to demonstrate front-end development concepts such as dynamic UI rendering, filtering, searching, and structured data handling.

## 🚀 Features

- 🎧 Play songs by clicking on song cards
- 🔍 Search songs by title
- 🎼 Filter songs based on genre
- 👨‍🎤 Filter songs based on artist
- 📊 Track number of times a song is played
- 🕒 Recently played songs (last 5 songs stored)
- 📱 Responsive grid layout for playlist display
- 🎯 "Now Playing" section showing current song

---

## 🛠️ Technologies Used

- HTML5 (structure)
- CSS3 (styling)
- TypeScript (logic and type safety)
- JavaScript (compiled output for browser)
